# flowlet 実装タスクリスト

## 方針
- 直近の実装体験をもとに、次に着手すべきタスクを優先度付きで整理する
- `Start.md` の方針である「実運用で見える課題」「説明可能な設計」「学習しやすい実装順」を反映する
- 実装、検証、記事化の切り口も合わせて残す

## 現状
- `Account`、`GoalBucket`、`Category / Subcategory`、`Transaction / Transfer / GoalBucketAllocation` の基本 CRUD と画面は一通りある
- CSV 取り込みは現状スコープ外
- 次は「本番で使って気になった改善」と「主要画面の実用性向上」を優先する

## 優先メモ
- P1 は本番利用時の視認性、導線、情報不足の解消を優先する
- P2 は日常利用で見る頻度の高い一覧や分析を強化する
- P3 は入力効率化や将来拡張を進める

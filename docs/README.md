# Docs Index

`docs/` は用途別に次の 4 つへ整理しています。

- `docs/project/`: 要件、実装タスク、実装計画
- `docs/specs/`: API 仕様、画面/機能の設計メモ
- `docs/ops/`: セットアップ、運用ルール、公開データ方針
- `docs/articles/`: 記事管理（一覧と本文）

## 追加時ルール

- API 仕様や設計は `docs/specs/` に配置する
- 実行順や計画、要求整理は `docs/project/` に配置する
- 手順や運用ポリシーは `docs/ops/` に配置する
- 記事本文は `docs/articles/` に配置する

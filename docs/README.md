# Docs Index

`docs/` は用途別に次の 4 つへ整理しています。

- `docs/project/`: 要件、タスク運用、実装計画
- `docs/specs/`: API 仕様、画面/機能の設計メモ
- `docs/ops/`: セットアップ、運用ルール、公開データ方針
- `docs/articles/`: 記事管理（一覧と本文）

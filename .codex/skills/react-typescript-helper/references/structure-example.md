## 曗懌

- `pages` 偼夋柺偺慻傒棫偰偵愱擮偡傞丅
- `features` 偼婡擻扨埵偱 UI丄API丄宆丄僼僅乕儉傪暵偠傞丅
- `shared` 偼杮摉偵嫟捠壔偱偒傞傕偺偩偗傪抲偔丅
- API 僋儔僀傾儞僩偼 `shared/lib` 偵抲偒丄婡擻暿 API 偼 `features/*/api` 偵抲偔丅
